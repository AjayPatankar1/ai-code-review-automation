# AI Code Review Automation

I observed repeated issues in AI-generated code such as hardcoded values, missing validations, and duplicated logic.

Based on this, I started creating a simple approach to convert common code review checks into small automated validations.

Current implementation includes:
- Basic rule definitions
- Simple script for detecting hardcoded values (initial version)
- Structure for API and UI validation

This is an initial version, and I am improving it step by step by adding real validations and automation.
